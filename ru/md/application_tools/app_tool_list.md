# Прикладной программный стек и инструменты разработки

## 1. Ollama
Ollama — это открытый кроссплатформенный инструмент для **локального развертывания больших языковых моделей (LLM)**. Он упрощает запуск, управление и инференс LLM в локальной среде. Пользователи могут с помощью простых команд разворачивать и вызывать предобученные модели (например, LLaMA и DeepSeek) на персональных устройствах, таких как ПК или edge-серверы, без зависимости от облачных сервисов или высокопроизводительных GPU.  
[Инструкция по Ollama](https://bianbu.spacemit.com/ai/ollama)

## 2. LocalAI
LocalAI — это **полный стек ИИ для локального запуска ИИ-моделей**. Он прост, эффективен и доступен, предоставляет API, совместимый с OpenAI, и позволяет запускать большие языковые модели (LLM), генерацию изображений, транскрипцию речи и другие ИИ-нагрузки на потребительском оборудовании, включая среды только с CPU, при этом сохраняя конфиденциальность и безопасность данных.  
[Инструкция по LocalAI](https://bianbu.spacemit.com/ai/localai)

## 3. Open-WebUI  
Open-WebUI (ранее Ollama WebUI) — это открытый, самохостируемый веб-интерфейс управления, созданный для **локальных или приватных развертываний LLM**. Его основная цель — предоставить пользовательский опыт уровня ChatGPT при поддержке офлайн-работы и широкой кастомизации.  
[Инструкция по Open-WebUI](https://bianbu.spacemit.com/ai/openwebui)

## 4. Node-RED  
Node-RED — это **визуальный инструмент программирования на основе потоков (flow-based)**, изначально разработанный инженерами IBM и ныне поддерживаемый open-source сообществом при поддержке JS Foundation. Он широко используется для событийно-ориентированной интеграции приложений, разработки ИИ-приложений, оркестрации API и автоматизации.  
[Инструкция по Node-RED](https://bianbu.spacemit.com/brdk/System_configuration/2.6_Docker_Usage/2.6.2_Node-RED_Usage)

## 5. Demo Docker
Demo Docker — контейнерный пакет, интегрирующий **ИИ-демо SpacemiT**. Он предоставляет веб-интерфейс для быстрого показа результатов инференса ИИ и позволяет познакомиться с возможностями ИИ без написания кода.  
[Инструкция по Demo Docker](https://bianbu.spacemit.com/brdk/System_configuration/2.6_Docker_Usage/2.6.4_DemoZoo_Usage)
