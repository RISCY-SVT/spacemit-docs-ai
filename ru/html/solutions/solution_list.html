ИИ чат-бот

Этот кейс демонстрирует ИИ чат-бота, реализованного через бесшовную интеграцию голосового модуля и больших языковых моделей. Он поддерживает логичные многоходовые диалоги и, благодаря глубокому пониманию естественного языка, предоставляет пользователю информацию и отвечает на вопросы.

Особенности проекта

  • Используемые модели: sensevoice, Deepseek R1, melotts
  • Поддержка языков: предоставлены примеры кода на Python
  • Возможности: сквозной ввод и вывод речи

Интеллектуальный AGV-робот сопровождения

Этот кейс демонстрирует, как AGV-робот глубоко интегрирует мультимодальное взаимодействие и автономную мобильность. Система поддерживает управление голосовыми командами и использует визуальное восприятие в реальном времени для точного следования за человеком. Большая языковая модель, используя механизм Function Call вместе с фреймворком ROS2, интеллектуально координирует движение робота. Решение органично объединяет восприятие окружения, планирование движения и голосовое взаимодействие, показывая потенциал интеллектуальных AGV в бытовых сервисах и промышленной автоматизации.

Особенности проекта

  • Используемые модели: qwen2.5-0.5B, YOLOv8
  • Поддержка языков: предоставлены примеры кода на Python
  • Возможности: голосовое управление, распознавание объектов на базе YOLOv8, интеллектуальное следование через коллбеки LLM

Умная розница с манипулятором

Этот кейс демонстрирует комплексное решение для умной розницы на базе платформы K1 и манипулятора Elephant. Система глубоко интегрирует интеллектуальное взаимодействие через Function Call, высокоточную детекцию объектов YOLOv8 и OCR-распознавание текста, обеспечивая полностью автоматизированный процесс от идентификации товара и сканирования штрихкода до оплаты и голосового взаимодействия. Решение повышает уровень автоматизации и операционную эффективность в розничных сценариях, а также показывает перспективы человеко-машинного взаимодействия в современной торговле.

Особенности проекта

  • Используемые модели: sensevoice, qwen2.5-0.5B, YOLOv8, Paddle OCR
  • Поддержка языков: предоставлены примеры кода на Python
  • Возможности: голосовое управление, коллбеки LLM, детекция объектов YOLOv8, распознавание текста

Манипулятор LeRobot

Этот кейс подробно описывает полный end-to-end процесс развертывания и запуска фреймворка LeRobot на платформе K1, поддерживая интеллектуальное управление манипулятором в реальных сценариях и разработку приложений. Материал охватывает сбор данных, обучение и развертывание модели ACT, а также тонкую настройку и распределенное развертывание модели SmolVLA.

Особенности проекта

  • Используемые модели: ACT, SmolVLA
  • Поддержка языков: предоставлены примеры кода на Python
  • Возможности: сбор данных, обучение модели и распределенное развертывание
